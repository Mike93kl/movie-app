# Requirements

 - >= Python 3.8
 - pip

# Installation locally:
 - Clone directory and CD into it
 - Run this command to copy .env file:
    + `cp .env.example` `.env`
 - Fill `.env` with the appropriate files
 - Run these commands:
    + `$> pip install pipenv (if not installed)`
    + `$> pipenv install`
    + `$> pipenv shell`
    + `$> python app.py`
